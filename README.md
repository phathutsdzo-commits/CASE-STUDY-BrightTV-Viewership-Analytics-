This project analyzes BrightTV's customer viewing behavior to provide insights that support the Customer Value Management (CVM) team's objective of increasing the company's subscription base. The analysis focuses on understanding user engagement, identifying viewing trends, and recommending strategies to improve customer retention and content consumption.

The main objectives of this case study were to:

Analyze user viewing patterns and subscription behavior.
Identify factors influencing content consumption.
Determine the most popular TV channels and viewing periods.
Segment viewers by demographics and activity.
Provide business recommendations to increase subscriptions and customer engagement.

The project was completed using the following approach:

Explored the user profile and viewership datasets.
Cleaned and transformed the data using SQL.
Created calculated fields such as:
Age Groups
Active vs Inactive Users
Time Buckets
Weekday vs Weekend Classification
Performed exploratory data analysis to identify trends and patterns.
Built interactive dashboards to visualize key performance indicators.
Generated business insights and recommendations based on the findings.
Key Insights

The analysis revealed several important insights:

Certain TV channels consistently attracted the highest number of viewers.
Viewership peaked during specific hours of the day.
Active users contributed the majority of total viewing hours.
Viewer engagement varied across age groups and regions.
Viewing behavior differed between weekdays and weekends.
High-performing channels present opportunities for targeted marketing and promotional campaigns.
